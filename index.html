Preformatted-Text | 'pre' element

Defination of 'pre' element:

'Pre' element reprensents preformatted text in html, it helps retain orginally written and formatted html. It can be used to display code snippets, formatted text, maintaining orginal spacing and line breaks.

'pre' element for line breaks and spacing:

With 'pre' elemet line breaks and spacing maintain as intended.

 
                *
                |
              _/o\_
             _/   \_
             \     /
             \     /
             \=[+]=/
            \= = = =/
            \= [+] =/
        ____\       /____
       |_________________| 
      / + + + + + + + + + \
     ||[][][][]   [][][][]||

With 'P' element, spacing and line breaks are lost

* | _/o\_ _/ \_ \ / \ / \=[+]=/ \= = = =/ \= [+] =/ ____\ /____ |_________________| / + + + + + + + + + \ ||[][][][] [][][][]||

'pre' element for JS code snippets

With 'p' element Js code shows on one line:

let favoriteFood = "Pizza"; let favoriteDrink = "Water"; let favoriteDessert = "Icecream"; let myFavStuff = [favoriteFood, favoriteDrink, favoriteDessert]; console.log(myFavStuff);

With "pre" element JS code retains its formatting:



let favoriteFood = "Pizza";
let favoriteDrink = "Water";
let favoriteDessert = "Icecream";

let myFavStuff = [favoriteFood, favoriteDrink, favoriteDessert];
console.log(myFavStuff);
